# HOW TO RUN FOR TONY

## SCRIPT: Get-PasswordNeverExpires-AZURE-ONLY.ps1

---

## WHAT IT DOES:
Gets all Azure AD users with password expiration disabled (DisablePasswordExpiration)
Creates CSV file on your Desktop
Opens CSV automatically

---

## HOW TO RUN:

1. Right-click PowerShell â†’ Run as Administrator

2. Run this command:
```powershell
.\Get-PasswordNeverExpires-AZURE-ONLY.ps1
```

3. Sign in to Azure AD when prompted

4. Wait for CSV to be created (1-2 minutes)

5. CSV will open automatically

6. Send CSV to Tony Schlak

---

## CSV CONTAINS:

- Display Name
- User Principal Name  
- Email
- Account Enabled (True/False)
- Password Policy (shows "DisablePasswordExpiration")
- Department
- Job Title
- Account Created Date
- User ID
