📋 How to Use

🔴 Run on CLIENT SERVER via Fleet Manager RDP — NOT on Jumphost!
  1. Connect to server via Fleet Manager → Remote Desktop
  2. Open Notepad on the client server
  3. Click the green Copy button for your server below
  4. Paste into Notepad → File → Save As → C:\patch.ps1
  5. Open PowerShell as Administrator
  6. Type C:\patch.ps1 → press Enter
  7. Wait for ALL DONE!

🔧 KB Fix — If KB Download Fails from S3

⚠️ If Read-S3Object gives Access Denied for the KB file — download it manually from AWS Console browser, then run these commands to move and install it.

Step 1 — Download KB from AWS Console:

  1. Go to AWS Console → S3 → ns2-il5-s4-support0002-sc-ssm-patching
  2. Navigate to patches/windows_patch/
  3. Click the windows10.0-kb5078752...msu file
  4. Click Download — it saves to Downloads folder

Step 2 — Move from Downloads and Install:

# Move KB from Downloads to Patches folder
Move-Item "C:\Users\Administrator\Downloads\windows10.0-kb5078752*.msu" "C:\Temp\Patches\kb.msu" -Force
Write-Host "KB file moved!" -ForegroundColor Green

# Install the KB
Write-Host "Installing KB5078752..." -ForegroundColor Yellow
Start-Process -FilePath "wusa.exe" -ArgumentList "C:\Temp\Patches\kb.msu /quiet /norestart" -Wait
Write-Host "KB5078752 Installed!" -ForegroundColor Green

# Verify KB installed
Get-HotFix | Where-Object {$_.HotFixID -eq "KB5078752"}

🖥️ SCRIPT A — Support002 Sidecar Server

Files are in patches/windows_patch/ — NO date folder.

⚠️ If KB download fails with Access Denied — use the KB Fix section above after Chrome/Edge/Defender install.
# ================================================
# IL5 PATCH SCRIPT — SUPPORT002 SIDECAR SERVER
# March 2026 — Run as Administrator
# ================================================

# Comment out if stuck:
# Import-Module AWSPowerShell

# Variables
$Bucket = "ns2-il5-s4-support0002-sc-ssm-patching"
$Prefix = "patches/windows_patch"
$Folder = "C:\Temp\Patches"
$Region = "us-gov-west-1"

# Create folder
New-Item -ItemType Directory -Path $Folder -Force | Out-Null
Write-Host "Folder ready!" -ForegroundColor Green

# Download 1 of 4 — Chrome
Write-Host "[1/4] Downloading Chrome..." -ForegroundColor Yellow
Read-S3Object -BucketName $Bucket -Key "$Prefix/GoogleChromeStandaloneEnterprise64.msi" -File "$Folder\chrome.msi" -Region $Region
Write-Host "  Chrome downloaded!" -ForegroundColor Green

# Download 2 of 4 — Edge
Write-Host "[2/4] Downloading Edge..." -ForegroundColor Yellow
Read-S3Object -BucketName $Bucket -Key "$Prefix/MicrosoftEdgeEnterpriseX64.msi" -File "$Folder\edge.msi" -Region $Region
Write-Host "  Edge downloaded!" -ForegroundColor Green

# Download 3 of 4 — Defender
Write-Host "[3/4] Downloading Defender..." -ForegroundColor Yellow
Read-S3Object -BucketName $Bucket -Key "$Prefix/mpam-fe.exe" -File "$Folder\defender.exe" -Region $Region
Write-Host "  Defender downloaded!" -ForegroundColor Green

# Download 4 of 4 — Windows KB5078752
# IF THIS FAILS — use KB Fix section above to download from browser
Write-Host "[4/4] Downloading Windows KB5078752..." -ForegroundColor Yellow
Read-S3Object -BucketName $Bucket -Key "$Prefix/windows10.0-kb5078752-x64_d19dce98bd2132ac8dcf4ae644ff36ec1ac7a754.msu" -File "$Folder\kb.msu" -Region $Region
Write-Host "  KB downloaded!" -ForegroundColor Green

Write-Host "ALL DOWNLOADED — INSTALLING NOW..." -ForegroundColor Cyan

# Install 1 of 4 — Chrome
Write-Host "Installing Chrome..." -ForegroundColor Yellow
Start-Process -FilePath "$Folder\chrome.msi" -ArgumentList "/quiet /norestart" -Wait
Write-Host "  Chrome installed!" -ForegroundColor Green

# Install 2 of 4 — Edge
Write-Host "Installing Edge..." -ForegroundColor Yellow
Start-Process -FilePath "$Folder\edge.msi" -ArgumentList "/quiet /norestart" -Wait
Write-Host "  Edge installed!" -ForegroundColor Green

# Install 3 of 4 — Defender
Write-Host "Installing Defender..." -ForegroundColor Yellow
Start-Process -FilePath "$Folder\defender.exe" -Wait
Write-Host "  Defender installed!" -ForegroundColor Green

# Install 4 of 4 — Windows KB
Write-Host "Installing KB5078752..." -ForegroundColor Yellow
Start-Process -FilePath "wusa.exe" -ArgumentList "$Folder\kb.msu /quiet /norestart" -Wait
Write-Host "  KB installed!" -ForegroundColor Green

Write-Host "================================================" -ForegroundColor Cyan
Write-Host "ALL DONE!" -ForegroundColor Green
Write-Host "================================================" -ForegroundColor Cyan

# Verify Defender
Write-Host "Defender check:" -ForegroundColor Yellow
Get-MpComputerStatus | Select-Object AntivirusSignatureLastUpdated, AntivirusSignatureVersion

# Verify KB
Write-Host "KB check:" -ForegroundColor Yellow
Get-HotFix | Where-Object {$_.HotFixID -eq "KB5078752"}
